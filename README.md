# 🛒 DigiTools

DigiTools is a simple and user-friendly e-commerce web application built with modern frontend technologies. Users can browse products, add items to the cart, remove items, and get real-time notifications for their actions.

---

## 🚀 Live Demo
🔗 [https://your-live-link.com](https://digi-tools-174.netlify.app/)

---

## 📌 Overview

DigiTools provides a smooth shopping experience with a clean UI. It focuses on simplicity and usability, allowing users to easily manage their cart and interact with products.

---

## ⚙️ Technologies Used

- React.js
- Tailwind CSS
- DaisyUI
- JavaScript (ES6+)
- React-Toastify
- Fake JSON Data

---

## ✨ Features

- 🛍️ Add products to cart
- ❌ Remove products from cart
- 🔔 Toast notifications (Add / Remove / Purchase)
- 🎨 Button hover effects
- 📦 Product card animations
- ⚡ Fast and responsive UI
